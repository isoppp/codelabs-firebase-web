## 学べること

- firebaseを使用したチャットクライアントの作成
- Firebase Realtime DatabaseとCloud Storageを使用してデータを同期させます。
- Firebase Authを使用してユーザを認証します。
- Firebase 静的ホスティングにWebアプリケーションをデプロイします。
- Firebase Cloud Messagingで通知を送信します。
